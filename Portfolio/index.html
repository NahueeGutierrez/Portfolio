Proyectos

Harry Potter

El proyecto "Harry Potter" es una aplicación web interactiva que ofrece a los usuarios una experiencia inmersiva al explorar un universo lleno de personajes fascinantes y poderosos hechizos. Desarrollado utilizando la biblioteca de JavaScript React, el proyecto se basa en la integración de dos APIs clave: una para recuperar datos relacionados con personajes y otra para obtener información detallada sobre diversos hechizos.

Repositorio en GitHub

Inicio de Sesión

La "Página de Inicio de Sesión" es una interfaz web diseñada para proporcionar a los usuarios un acceso seguro y eficiente a una plataforma exclusiva. Desarrollada con tecnologías web estándar, esta página combina HTML para la estructura, CSS para el diseño visual y JavaScript para una interactividad suave.

Repositorio en GitHub

Calculadora Street Fighter

La "Calculadora Street Fighter" es una experiencia única que combina la funcionalidad práctica de una calculadora con la emoción y la estética del universo de Street Fighter. Desarrollada con tecnologías web estándar, como HTML, CSS y JavaScript, esta calculadora te invita a realizar cálculos mientras te sumerges en el mundo de los combates callejeros
